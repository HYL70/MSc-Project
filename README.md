# Causal Emergence in Two-dimensional XY Model
This project collects the Python files I used for my MSc Project. `examples/demos.ipynb` shows how to use these files to compute the quantities in my thesis.

It includes:
  1. Simulation of XY model in `src/XYModel`.
  2. Methods to detect causal emergence in `src/Causal_Emergence_Quantity`.

**This project requires a toolkit from https://github.com/jlizier/jidt?tab=readme-ov-file to compute mutual information.**
